quickHull

  • quickHull(points: Vector[]): Facet[]
    • Defined in hull/QuickHull.ts:181

    QuickHull implements the algorithm of the same name, based on the original paper by Barber, Dobkin and Huhdanpaa. We're not interested in 0- or 1-dimensional cases (the latter can simply be sorted points). QuickHull returns a set of indices into the original point list so we can map it to a different original ata set (fe: points may be a mapping for position vectors on some scene graph object).
